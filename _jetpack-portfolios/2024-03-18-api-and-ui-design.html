Developing AI & ML algorithm is importtant. But using the models to build fancy AI & ML demo to show how they solve problem is also critical. As a algorithm developer, we need a tool to quick build a prototype.

Resources

  • Streamlit: easy to use and build fancy demo. Code style and format is Python. I have used it to build a chat to integrate LLM, TTS, and others.

  • Gradio.app: Huggingface demo use it. But have not try

  • FastAPI: Deploy your AP & ML models as a service. Very good to use

  • Server Sent Events: it is useful when building streaming service, e.g. streaming LLM response, TTS, video to frontend.

  • Web search (free): Google search API: Wrap Google web search

  • Web search (free): Duckduckgo search API

  • Altair: declarative statistical visualization library for Python
